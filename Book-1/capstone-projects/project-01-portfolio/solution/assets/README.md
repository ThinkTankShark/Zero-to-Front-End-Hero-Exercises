# Assets Folder

## Resume File

Place your resume PDF here:

**File name:** `resume.pdf`

### Creating Your Resume

Your resume should include:

1. **Contact Information**
   - Name
   - Email
   - Phone (optional)
   - Location (City, State)
   - Links (Portfolio, GitHub, LinkedIn)

2. **Summary/Objective**
   - 2-3 sentence overview
   - Highlight key skills
   - Career goals

3. **Technical Skills**
   - Front-End: HTML5, CSS3, Responsive Design
   - Tools: Git, VS Code, Chrome DevTools
   - Currently Learning: JavaScript, React, etc.

4. **Projects** (3-5 best projects)
   - Project name
   - Technologies used
   - Brief description
   - Link to live site and code

5. **Education**
   - Degree/Certification
   - Institution
   - Graduation date
   - Relevant coursework

6. **Experience** (if applicable)
   - Previous work
   - Volunteer projects
   - Freelance work

### Resume Tips for Developers

✅ **Do:**
- Keep it to 1 page (2 max if experienced)
- Use a clean, readable format
- Include links to your work
- Quantify achievements when possible
- Proofread carefully (no typos!)
- Use action verbs (Built, Created, Designed)
- Tailor to each job application

❌ **Don't:**
- Use fancy graphics or colors (unless design role)
- Include photo (in USA)
- List every technology you've touched
- Use generic descriptions
- Forget to update it regularly

### Exporting as PDF

From Google Docs or Word:
1. File → Download → PDF
2. Name it `resume.pdf`
3. Place in this `assets/` folder
4. Test the download link works

---

**Important:** Keep your resume updated! Update it whenever you:
- Complete a new project
- Learn a new skill
- Finish a course
- Get new experience
