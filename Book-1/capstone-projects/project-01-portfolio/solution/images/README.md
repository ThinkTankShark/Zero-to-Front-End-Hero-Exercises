# Solution Images

This solution uses placeholder images. In a real portfolio, you would replace these with:

## Required Images

### 1. Profile Photo (`profile.jpg`)
- **Location:** Root of `images/` folder
- **Recommended size:** 400x400px (square)
- **Format:** JPG or PNG
- **Usage:** About section profile picture
- **Placeholder:** Use any professional headshot

### 2. Project Screenshots
**Location:** `images/projects/` folder

- `project1.jpg` - E-Commerce website screenshot
- `project2.jpg` - Restaurant website screenshot
- `project3.jpg` - Weather dashboard screenshot

**Recommended specs:**
- **Size:** 1200x675px (16:9 aspect ratio)
- **Format:** JPG (optimized)
- **File size:** < 300KB each
- **Quality:** Show the full project interface

## How to Add Your Own Images

### Option 1: Use Your Own Photos
1. Take a professional photo or use an existing one
2. Resize to 400x400px
3. Name it `profile.jpg`
4. Place in `images/` folder

### Option 2: Use Placeholder Images

For testing, you can use placeholder services:

```html
<!-- Profile image -->
<img src="https://via.placeholder.com/400" alt="Your name">

<!-- Project images -->
<img src="https://via.placeholder.com/1200x675" alt="Project screenshot">
```

### Option 3: Free Stock Photos

Get professional photos from:
- [Unsplash](https://unsplash.com/)
- [Pexels](https://www.pexels.com/)
- [Pixabay](https://pixabay.com/)

## Image Optimization

Before adding images, optimize them:

1. **Resize** to exact dimensions needed
2. **Compress** using:
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   - [ImageOptim](https://imageoptim.com/)
3. **Convert** to WebP for better compression (optional)
4. **Test** file size (aim for < 300KB per image)

## Project Screenshots Tips

### How to Get Good Screenshots

1. **Full project view** - Show the complete interface
2. **Clean browser** - Hide bookmarks bar
3. **Realistic content** - Use real or realistic data
4. **Good lighting** - Use proper contrast
5. **Multiple devices** - Show mobile views too

### Tools for Screenshots

- **Browser DevTools** - Built-in screenshot tools
- **Awesome Screenshot** - Browser extension
- **CloudApp** - Desktop screenshot tool
- **Figma/Photoshop** - For mockups
