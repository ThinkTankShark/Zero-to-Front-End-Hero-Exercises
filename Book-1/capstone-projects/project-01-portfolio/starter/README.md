## Getting Started

1. **Customize the Content**
   - Replace all `[Your Name]` placeholders with your actual name
   - Add your own bio and project descriptions
   - Update links to your GitHub, LinkedIn, etc.

2. **Add Your Images**
   - Add your profile photo as `images/profile.jpg`
   - Add project screenshots in `images/projects/`
   - Optimize all images before adding

3. **Style with CSS**
   - The CSS file has a solid foundation
   - Complete the styles for each section
   - Add your own custom touches

4. **Customize Colors**
   - Modify CSS variables in `:root` to match your brand
   - Choose a color palette that represents you
