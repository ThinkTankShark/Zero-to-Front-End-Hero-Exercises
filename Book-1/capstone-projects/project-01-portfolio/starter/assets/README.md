# Assets Folder

Place your resume PDF in this folder.

**File name:** `resume.pdf`

## Quick Resume Tips

Your resume should include:
- Contact information
- Summary/objective
- Technical skills (HTML, CSS, etc.)
- Projects (3-5 best ones)
- Education
- Experience (if applicable)

Keep it simple, clean, and 1 page if possible!

For templates and tips, see the solution folder's assets/README.md
