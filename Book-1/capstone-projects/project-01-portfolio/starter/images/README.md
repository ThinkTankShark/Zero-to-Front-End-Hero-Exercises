# Images Folder

Place your images here:

## Required Images

### Profile Photo
- **File:** `profile.jpg` or `profile.png`
- **Size:** 400x400px minimum (square)
- **Format:** JPG or PNG
- **File size:** Keep under 200KB
- **Tip:** Use a professional, high-quality headshot

### Project Screenshots
Place in `projects/` subfolder:
- `project1.jpg` - Screenshot of your first project
- `project2.jpg` - Screenshot of your second project
- `project3.jpg` - Screenshot of your third project

**Recommended size:** 1200x800px (3:2 aspect ratio)
**Format:** JPG (compressed) or WebP
**File size:** Keep each under 300KB

## Image Optimization Tips

1. **Resize images** before adding them (don't let CSS do it)
2. **Compress images** using tools like:
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   - [ImageOptim](https://imageoptim.com/) (Mac)
3. **Use WebP format** for better compression (with JPG fallback)
4. **Add lazy loading**: `<img loading="lazy">`

## Free Stock Photos

If you need placeholder images:
- [Unsplash](https://unsplash.com/)
- [Pexels](https://www.pexels.com/)
- [Pixabay](https://pixabay.com/)
